Aventuriers du Rails Europe

Maquette du Projet

Interface finale

Nous devions créer le jeu de société "Les Aventuriers du Rails Europe" en JAVA en Binôme. Le projet était divisé en 3 partie :

  • Le code
  • Les graphes
  • L'interface

Pour la partie code nous avions une interface fonctonnel et nous devions faire le code du jeu en prenant en compte les règles du jeu et des tests qui nous ont été fournis. Pour la partie graphe, nous devions utiliser diférents algorithmes afin de calculer le court trajet, le plus long, etc... Enfin, pour la partie de l'interface, nous avions un code fonctionnel et on devais réaliser l'interface fonctionnelle du jeu. Cela va être difficile de vous illustrer le code donc je vous montre que la partie de l'interface.
